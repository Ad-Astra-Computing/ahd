Framework

AI-generated design defaults to average. It smooths edges, centers text, and copies what it has already seen. It produces slop.

AHD is a framework for turning design intent into rules a model can follow. It is not a model itself. It is a compiler, a linter, and an evaluation harness for running design prompts against foundation models and measuring the result. It is a way to demand better.

ahd eval-live swiss-editorial --brief briefs/landing.yml --models  --n 10

What ships

  • Brief compiler
  • 28-rule linter
  • Eval harness
  • Vision-critic scaffold
  • MCP server
  • Editor plugins
  • Eight tokens

What is gated

  • Live frontier-model calls (needs API keys)
  • Live vision critique (needs multimodal key + screenshot pipeline)
  • Standalone npm packages for the editor plugins
  • Additional tokens
