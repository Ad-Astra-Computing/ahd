# Website (future)

Planning doc for the eventual public site at `ahd.adastra.computer` (or standalone). **Nothing here is implemented yet.** The rule: every pixel on this site is rendered by AHD against its own tokens. If we ship a page that would fail `ahd lint`, the site itself has a bug.

## Dogfooding contract

- Every page built from a brief in `briefs/site/*.yml` compiled against a token in `tokens/`.
- CI gate: `ahd lint site/dist/**/*.html site/src/**/*.css` must pass on every push.
- Every major visual demonstrates a token. No generic "hero with gradient" page anywhere.
- `ahd critique` runs against PR screenshots as a comment.

## Proposed pages

1. **Home.** Masthead rendered from `swiss-editorial`. Headline measured numbers (current: Claude 100%, Mistral 62%, Llama 3.3 −150%). Four-piece value prop. Prior art row. CTA: GitHub + docs.
2. **The Taxonomy.** All 39 tells, rendered inline as small example images — each tell paired with a "slop exemplar" and a "counter-example" generated by AHD. Click a tell → see the lint rule that catches it, the token that forbids it, and an eval sample where it fired / didn't fire.
3. **Interactive timeline of AI-era design aesthetics.** *(dogfooded against `editorial-illustration` and `swiss-editorial`.)* A horizontal scroll from 2017 → 2026 showing the visual house-style of each era:
   - 2017–2019: Corporate Memphis / Alegria (Facebook, Slack, Shopify)
   - 2019–2021: Stripe-era gradient + 3D blob
   - 2020–2022: shadcn / purple-blue gradient / rounded-2xl / Inter
   - 2022–2024: Neubrutalism (Gumroad 2021 lineage)
   - 2023–2025: Editorial Neo-Modernism (Linear, Cash App for Business)
   - 2023–2026: Post-Digital / terminal aesthetics (Are.na lineage, dev tools)
   - 2024–2026: Glossy 3D revival (Apple Intelligence)
   - 2025–2026: Y2K / Frutiger Aero revival
   - 2025–2026: Anti-design / deconstructed zine
   Each era: one real screenshot exemplar, one caption, one `tokens/<id>.yml` pointing at the AHD token that serves that direction.
4. **Eval live.** A published, always-current leaderboard: latest `ahd eval-live` run across the full seed brief corpus. Updated by the scheduled GH Action weekly. Every cell linkable back to the exact run manifest and the sampled HTML.
5. **Docs.** Rendered from `docs/*.md` via a minimal static site generator that consumes AHD tokens for typography and grid.
6. **Contribute a token.** Form → PR template → GitHub Action that validates the new token against the schema.

## Anti-checklist (what the site must not become)

- No centred hero with pill badge, headline, two CTAs
- No three-feature-card row
- No purple-to-blue hero gradient
- No rounded-2xl everywhere
- No emoji bullets
- No Inter as only face
- No "Build the future of design" copy
- No stock-team-laptop hero photo
- No Corporate Memphis illustration

If it would fail `ahd lint`, it doesn't ship.

## Stack sketch (not committed)

- **Astro** or **11ty** for static output — both emit clean HTML the linter can score.
- **Playwright** for visual regression + the critique pipeline.
- **Cloudflare Pages** for hosting; Workers for any dynamic leaderboard endpoints.
- Deployment via the same `nix develop` shell as the flake.
